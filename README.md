# **Video to Audio Converter**

A fast and simple web tool to extract MP3 audio from video files entirely in your browser.

---

## **Features**

* **Local Processing:** All conversion happens inside your browser. Your video files are never uploaded to a server, ensuring 100% privacy.
* **Batch Support:** Convert multiple videos at once to save time.
* **High Quality:** Extracts audio using high-quality variable bit rate settings.
* **Clean & Responsive:** A minimalist interface that supports Dark Mode and works on any device.

---

## **How to Use**

1.  **Open the Tool:** Navigate to the live URL.
2.  **Select Files:** Click "Choose Files" and select the video files (e.g., MP4, MOV) you want to convert.
3.  **Process:** The tool will automatically start converting them one by one.
4.  **Download:** Once a file is finished, a "Download MP3" button will appear. Click it to save your audio.
